## Quick Start

1. **Clone the repository:**
    ```bash
    git clone https://github.com/Raja-Ramees/sysjoyprobe.git
    ```

2. **Navigate to the project directory:**
    ```bash
    cd sysjoyprobe
    chmod u+x sysjoyprobe.sh
    ```

3. **Run the script:**
    ```bash
    ./sysjoyprobe.sh
    ```
